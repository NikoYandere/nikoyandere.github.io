Yanix Launcher:
Join Yandere Simulator's Linux/Mac Revolution

The Yanix Launcher is an independent initiative created to run Yandere Simulator on Linux operating systems. It is neither supported nor developed by YandereDev. Please do not request a Windows port—this launcher is specifically optimized for Linux/Mac OS. Similarly, please avoid asking for universal package formats like .deb, .appimage, or .snap, as we provide the launcher as a Python 3 script (.py) and UCN Package (.ucn) for simplicity.

The Yanix Launcher runs the game using WINE, and requires WINE 8.0 or newer for proper functionality. While most major display bugs—like fullscreen issues—have been fixed in the Unity 6 Build, some letter display problems persist. Fixing these would require YandereDev's investigation, as this is beyond the scope of the launcher. Keep in mind that WINE may still encounter display-related issues, and in the future, a switch to GE-Proton might be considered.

This launcher is open-source, and both the GitHub repository and installer are available. Yanix has been developed and tested primarily for Linux/Mac, and while a FreeBSD build is unlikely to be released, you are welcome to fork the repository to adapt it for your preferred Unix/Linux distribution.

Bug reports such as "Letters are not showing for me!" will not be addressed. These issues are entirely YandereDev’s responsibility, not ours.

Frequently Answered Questions

Does this launcher support YandereDev? No.
Is this a port? Not exactly. It uses the WINE version you already have installed, making it a semi-port — it doesn’t modify the game code but uses a compatibility layer (WINE) to run Yandere Simulator on Linux.
Is it available for Mac? Yes, version 0.2.2.4 introduced Mac optimizations.
Is Yanix stolen? No. Yanix is basically a WINE GUI built using a Tkinter window with buttons, custom background, and icon. It's fully original and open-source.
