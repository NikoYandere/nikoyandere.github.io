The Definitive Yandere Simulator Launcher for Linux & macOS

Yanix Launcher is an open-source project dedicated to bringing Yandere Simulator to the Linux and macOS communities. Using WINE (version 8.0+ recommended), it provides a streamlined way to install, update, and play the game.

While most major display issues have been resolved, some font rendering bugs may persist. You can often fix these by running winetricks allfonts in your terminal. This project is community-driven and independent of the official game developer.

Windows Version (Akademi Launcher): A native Windows version is in development under the name Akademi Launcher. It is not yet on par with the Linux/Mac versions. Track its progress on the Akademi Launcher GitHub.

For UCN Package Manager Users: The .ucn package requires the UCN Package Manager. After installation, remember to run: chmod +x ~/.ucndata/packages/org.nikoyandere.yanix-launcher/yanix-launcher/binary/yanix-launcher

Minimum System Requirements

  • Kernel: 5.10 or newer
  • Processor: Intel Core i5 or equivalent AMD processor (Modern dual-core or older quad-core CPU)
  • RAM: 6 GB
  • Graphics: Intel HD Graphics, Iris Xe, Vega 3-8, RDNA2... (Must be DirectX 11 compatible)
  • VRAM/GTT: 4+ GB dedicated graphics memory (for best performance)
  • DirectX: Version 11 (handled by WINE)
  • Storage: Around 5 GB of free space

Frequently Asked Questions (FAQ)

Is this launcher an official Yandere Simulator product?

No, Yanix Launcher is an independent, community-driven project and is not affiliated with or endorsed by YandereDev.

Is this a native port of the game?

Not exactly. Yanix Launcher uses the WINE compatibility layer to run the original Windows version of Yandere Simulator on Linux and macOS. It does not modify the game's source code.

Is Yanix Launcher available for macOS?

Yes, macOS support and optimizations were introduced starting with version 0.2.2.4.

Is the code for Yanix Launcher original?

Yes. The launcher is a custom GUI built with Qt5 to launch Yandere Simulator. The full source code is original and available on our GitHub repository.

Where can I find the source code for this website?

The source code for this landing page is also open-source and can be found on GitHub.
